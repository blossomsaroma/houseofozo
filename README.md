# House of Ozo Video Website

A simple, elegant Next.js website featuring a video background with company branding.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your assets to the `public` folder:
   - `logo.png` - Your company logo (recommended size: 150x150px or larger)
   - `video.mp4` - Your background video

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
npm start
```

## Deployment

This project is ready to be deployed on platforms like:
- Vercel (recommended for Next.js)
- Netlify
- Any platform that supports Next.js

Simply connect your repository and the platform will handle the build process.

## Features

- Responsive design
- Video background with autoplay
- Company logo at the top
- Elegant text overlay with:
  - Main heading: "A gift from memory returns"
  - Subheading: "A trace of the familiar. A touch of the unknown."
  - "Coming soon" message
